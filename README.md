Project Title Kumo’s Portfolio — Megan Villadelgado Nazareth BSIT 3-2

Short Description A personal web portfolio showcasing my background, skills, projects, and 
creative work as a BSIT student from the Polytechnic University of the Philippines. Designed 
with a retro pixel aesthetic, this website reflects my passion for both technology and art, 
using parallax effects and interactive design to bring my story to life.

How to Run Locally Clone this repository:git clone https://github.com/clouded000/FINAL-PORTFOLIO_NAZARETH.git 
Navigate into the folder:cd FINAL-PORTFOLIO_NAZARETH Open index.html in your browser (no server setup required). 
You can also use a simple live server extension in VS Code for the best experience.
